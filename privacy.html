プライバシーポリシー

最終更新日: 2026年4月30日

Packer（以下「本アプリ」といいます）は、Daisuke Nakagawa（以下「当方」といいます）が提供するアプリです。当方は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本アプリがどのように情報を収集、使用、保護するかを説明します。

1. 収集する情報

1.1 アカウント情報

  • メールアドレス
  • 表示名（任意）
  • プロフィール画像（任意）

1.2 旅行情報

  • 旅行名
  • 目的地
  • 旅行日数
  • 旅行タイプ
  • 季節
  • 出発日・帰国日
  • メモ欄に入力された情報

1.3 持ち物リスト情報

  • ユーザーが作成または編集した持ち物リストの内容
  • チェック状態
  • カテゴリ情報

1.4 自動的に収集される情報

  • デバイス情報（OSバージョン、デバイスモデル）
  • アプリの利用状況（クラッシュレポート、パフォーマンスデータ）
  • IPアドレス

2. 情報の利用目的

収集した情報は、以下の目的で使用されます：

2.1 サービスの提供

  • アカウントの作成と管理
  • 旅行情報と持ち物リストの保存と同期
  • AIによる持ち物リスト生成
  • 出発前のプッシュ通知送信

2.2 サービスの改善

  • アプリの機能改善と新機能開発
  • バグの修正とパフォーマンスの向上
  • ユーザー体験の最適化

2.3 セキュリティ

  • 不正アクセスの防止
  • アカウントの保護
  • 利用規約違反の検出

3. 情報の共有

3.1 第三者への提供

本アプリは、以下の場合を除き、ユーザーの個人情報を第三者に提供しません：

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合

3.2 サービスプロバイダー

本アプリは、以下のサービスを利用しています：

  • Supabase: データベースと認証
  • Anthropic Claude API: 持ち物リスト生成AI

これらのサービスプロバイダーは、本アプリがサービスを提供するために必要な範囲でのみ情報にアクセスします。

3.3 AI機能における第三者AIサービスへの送信

本アプリは、AIによる持ち物リスト生成機能を提供するため、ユーザーが入力した旅行情報（旅行名、目的地、旅行日数、旅行タイプ、季節）をAnthropic Claude APIに送信します。

送信された情報は、持ち物リストを生成する目的でのみ使用されます。Anthropic Claude APIに送信された入力・出力データは、Anthropicのデータ保持ポリシーに基づき、最大30日間保持される場合があります。

本アプリは、AI機能の利用前に、送信されるデータ内容、送信先、利用目的をアプリ内で説明し、ユーザーの同意を取得します。ユーザーが同意しない場合、AI機能は利用できず、当該情報はAnthropic Claude APIに送信されません。

4. データの保存と削除

4.1 データの保存期間

  • アカウント情報: アカウント削除まで
  • 旅行情報: ユーザーが削除するまで
  • 持ち物リスト: ユーザーが削除するまで
  • Anthropic Claude APIに送信された入力・出力データ: Anthropicのデータ保持ポリシーに基づき最大30日間保持される場合があります

4.2 データの削除

ユーザーは、いつでもアプリ内から以下のデータを削除できます：

  • 個別の旅行と関連する持ち物リスト
  • アカウント全体（すべてのデータが削除されます）

5. データのセキュリティ

本アプリは、以下のセキュリティ対策を実施しています：

  • SSL/TLS暗号化による通信の保護
  • Row Level Security（RLS）によるデータベースアクセス制御
  • 定期的なセキュリティアップデート
  • パスワードの暗号化保存

6. 子どもの利用

本アプリは、16歳未満の子どもを対象としていません。16歳未満の子どもから意図的に個人情報を収集することはありません。

7. プライバシーポリシーの変更

本プライバシーポリシーは、法令の変更やサービスの改善に伴い、予告なく変更される場合があります。重要な変更がある場合は、アプリ内で通知します。
